Getting Started with the MATLAB Central Interface for MATLAB

Description

The Community API allows developers to programmatically retrieve data from MATLAB Central. The API offers a single resource to search across different MATLAB Central areas including MATLAB Answers, File Exchange, Blogs, Cody, Highlights and Contests. Use this toolbox for easy integration with the Community API from your MATLAB scripts and apps.

System Requirements

Requires MATLAB® release R2017b or newer

Functions

  • SearchApi.search - Search MATLAB Central areas for content matching specified query. Areas you can search include MATLAB Answers, File Exchange, Blogs, Cody, Community Highlights, and Community Contests. The scope parameter lets you restrict the request to specific MATLAB Central areas. The request accepts several other parameters to help narrow your search and create complex queries.

Usage

Download MATLAB_Central_Interface_For_MATLAB.mltbx into your workspace.

Examples

The following example demonstrates calling the Search API to retrieve MATLAB Answers for content matching the query term 'plotting' in the last one year. The example also demonstrates the use of all optional query parameters supported by the API.
Call the Search API using the search method passing in values for all optional parameters
query = "plotting";
scope = "file-exchange,matlab-answers";
tags = "legend";
created_before = datetime('12-Mar-2023 11:00:00');
created_after = datetime('06-Mar-2022 11:00:00');
sort_order = "created asc";
page = 3;
count = 15;
 
[searchResults, statusCode, headers] = SearchApi.search(query=query,scope=scope,tags=tags,created_before=created_before,created_after=created_after,sort_order=sort_order,page=page,count=count);
Extract the items and total_found objects from the response
totalFound = searchResults.total_found;
items = searchResults.items
